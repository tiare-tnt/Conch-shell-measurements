# Conch-shell-measurements
This project aims to determine whether the conch population is adult or juvenile to assess population dynamics across multiple beaches in the Andros Islands, supporting research in order to maintain sustainable fisheries.
Two graphs are created to visualize the data. Graph #1 is to identify the average length (cm) per site in the Andros islands. Graph #2 is to identify how many adults were present during sampling and measuring based on length and shell thickness.
